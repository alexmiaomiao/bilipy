# 多线程下载B站视频

Installing:
```bash
pip3 install bs4
pip3 install requests
pip3 install progressbar
brew install ffmpeg
```

Get started:
```bash
chmod +x bilipy.py
./bilipy https://www.bilibili.com/video/av11268454
```
